## element-wise product

每个元素对应位置相乘

```py
import numpy as np
w=np.ones(2,2)
w1 = w*3
w2[:,1] = 2

y = w1*w2
y = multiply(w1,w2)
```

## 矩阵乘法

注意矩阵乘法是按‘行’乘‘列’得到新矩阵的元素,即[n,m]*[m,w]=[n,w]

```
y1=np.dot(w1,w2)
y2=np.matmul(w1,w2)
y3=w1@w2
```
如果想按‘行’乘‘行’得到新矩阵元素，在np中称为inner

```
y = np.inner(w1,w2)
```
